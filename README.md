Демон Лапласа по определению - мысленный эксперимент, который предполагает существование такой сущности, которая может просчитывать движение каждого объекта. Если говорить про данное определение, то думаю, что вероятность его существования = 0, т.к. не верю в мистику. Если же говорить о явлении в целом - "просчитывать положение каждого объекта в будущем", то тут уже интереснее. Вообще само понятие не кажется НЕВОЗМОЖНЫМ. Что если собрать такой супер-компьютер, которому хватит мощности, чтобы просчитать передивижение атомов и предугадать их положение через секунду/минуту/час. По сути это и есть предсказание будущего, только основанное на математике и физике.

Функция - если говорить про программирование, то это обособленный участок кода, который может иметь вход и выход. Функция имеет название, которое (за исключение эджкейсов) отражает сущность того, что выполняет принадлежащий ей код.
Из моей дейли лайф функциями можно назвать:
Позавтракать (принимаем на вход пищу, сам код ее "поглощает", на выход имеем джоули)
Умыться (на вход вода, код делает garbage collect на моем лице))), на выходе обновленное лицо)
Вспомнить план на день (на вход ничего, код лезет в бд (воспоминания), берет оттуда записи событий, у которых date == сегодня)
